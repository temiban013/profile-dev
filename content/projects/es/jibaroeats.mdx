## Descripción del Proyecto

Este sitio web de portafolio muestra mi trabajo y experiencia utilizando tecnologías web modernas. Cuenta con:

- Arquitectura App Router con Next.js 14
- TypeScript para seguridad de tipos
- Contentlayer para gestión de contenido
- Soporte multilingüe (Inglés/Español)
- Diseño responsivo con Tailwind CSS
